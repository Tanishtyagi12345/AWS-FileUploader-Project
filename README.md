# AWS File Uploader Project

A simple file uploader using AWS S3, Lambda, API Gateway, and DynamoDB. Built using Python, HTML, and JavaScript.

## Features
- Presigned S3 URL generation
- Secure file upload
- DynamoDB metadata storage

## Tech Stack
- AWS S3
- AWS Lambda
- API Gateway
- DynamoDB
- Python (Boto3)
- HTML + JavaScript
